Hilfe

Der N-Brief hilft mit Textbausteinen einen Arztbrief zusammenzustellen.
Das Programm befindet sich in Entwicklung.
Die Texte sind in Zusammenarbeit entstanden. Dirk Bahner verfasste die Texte MS, BSV und BPLS. Den Text Klinischer Befund verbesserten Csilla Vincze, Dirk Bahner und Axel Saure. Marc Hennies hat viele gute Verbesserungsvorschläge zur Sprache der Texte gemacht.
Zum Nachschlagen eines Wortes ist www.duden.de eine gute Idee. Für die neurologische Terminologie sollen wir uns nach der DGN richten.
Die Knöpfe mit der Umrandung fügen Textbausteine in den Textbereich hinzu oder sind die Eingabe- bzw. die Tab-Taste.
Der Typ-korrekt-Knopf verschönert typografisch:
1. Wenn Leerzeichen, nur ein Leerzeichen
2. Kein Leerzeichen vor . , ? : )
3. Kein Leerzeichen nach (
4. Ein Leerzeichen nach . , : wenn keine Zahl folgt
5. Ein Leerzeichen vor %
6. Ein Leerzeichen nach einer Zahl, wenn ein Buchstabe folgt
7. 10.30 Uhr wird 10:30 Uhr
Grauer bzw. schwarzer Knopf bedeutet, dass der Textbaustein noch in Vorbereitung ist.
Bitte lesen Sie den eingefügten Text im Textbereich kritisch und bearbeiten Sie ihn sorgfältig.
Wer lieber diktiert, kann die Texte als Anleitung oder Inspiration nutzen.
Mit dem Knopf "Alles kopieren" wird der Inhalt des Textbereichs in die Zwischenablage kopiert. Dann kann er z. B. ins Orbis eingefügt werden.
Klicken Sie auf die Knöpfe Mann oder Frau. So wird das richtige Geschlecht des Patienten in die Texte eingefügt.
Das Programm verändert nichts an Ihrem Rechner, es verbindet sich nicht mit dem Internet, es braucht keine Installation. Es besteht aus einer einzigen einfachen Text-Datei, die HTML-, CSS- und Javascript-Code enthält. Dieser Code wird von Ihrem Browser interpretiert. Wenn Sie HTML-, CSS- und Javascript kostenfrei und ohne Anmeldung lernen möchten, besuchen Sie: w3schools. Der N-Brief funktioniert mit einem neuen Browser, am besten mit Firefox, Version mindestens 45.
Das Programm soll nützlich sein, sagen Sie bitte, was Sie sich wünschen.

Tastenkombinationen

Die folgenden Tastenkombinationen gelten in den meisten Windows-Anwendungen.
Sie helfen bei der Textbearbeitung.
Sie gelten auch für das weiße Textfeld im 'N-Brief'.

Kopieren, Einfügen, Finden

'Strg' + C : Die Auswahl kopieren
'Strg' + X : Die Auswahl ausschneiden
'Strg' + V : Die Auswahl einfügen
'Strg' + F : Suchen

Rückgängig

'Strg' + Z : Die vorherige Aktion rückgängig machen
'Strg' + Y : Die zurückgenommene Aktion wiederherstellen

Cursor-Bewegung

'Strg' + 'Pfeil rechts' : Cursor ein Wort nach rechts
'Strg' + 'Pfeil links' : Cursor ein Wort nach links
'Pos1' : Cursor zum Beginn der Zeile
'Ende' : Cursor zum Ende der Zeile
'Strg' + 'Pos1' : Cursor zum Beginn des Textes
'Strg' + 'Ende' : Cursor zum Ende des Textes
'Tab' : Die Tab-Taste funktioniert nun normal.

Auswählen

'Strg' + A : Den gesamten Text auswählen
'Umschalt' + 'Pfeil rechts' : Eine Stelle rechts vom Cursor auswählen
'Strg' + 'Umschalt' + 'Pfeil rechts' : Das Wort rechts vom Cursor auswählen
'Umschalt' + 'Pos1' : Bis zum Beginn der Zeile auswählen
'Umschalt' + 'Ende' : Bis zum Ende der Zeile auswählen
'Strg' + 'Umschalt' + 'Pos1' : Bis zum Beginn des Textes auswählen
'Strg' + 'Umschalt' + 'Ende' : Bis zum Ende des Textes auswählen
Doppelklick auf ein Wort : Das Wort auswählen
